# Setting Up
To set up, you can execute setup.sh

OR


Run commands from root StaticWeb directory:

1. Run `npm install`
2. Run `bower install`
3. Run `gulp`

# Running Local Server
NOTE: this only serves static web content

Run command from root StaticWeb directory: `gulp serve`
